# 🚗 Cars Mileage Prediction using Machine Learning

## 📌 Project Overview
This project predicts the mileage (MPG) of cars using the Linear Regression algorithm. It includes data understanding, data cleaning, exploratory data analysis (EDA), visualization, feature selection, model building, and prediction.

## 📂 Dataset
- Dataset: mtcars.csv
- Total Records: 32
- Features:
  - mpg
  - cyl
  - disp
  - hp
  - drat
  - wt
  - qsec
  - vs
  - am
  - gear
  - carb

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📊 Project Workflow

1. Import Libraries
2. Load Dataset

4. Data Cleaning
5. Exploratory Data Analysis (EDA)
6. Data Visualization
7. Feature Selection
8. Train-Test Split
9. Build Linear Regression Model
10. Model Evaluation

## 📈 Machine Learning Algorithm
- Linear Regression

## 📋 Model Evaluation
